# svc ls
```console
$ copilot svc ls
```

## コマンドの概要

`copilot svc ls` は特定の Application のすべての Service を一覧表示します。

## フラグ

```
  -a, --app string   Name of the application.
  -h, --help         help for ls
      --json         Optional. Output in JSON format.
      --local        Only show services in the workspace.
```

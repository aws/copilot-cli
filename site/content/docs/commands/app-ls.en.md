# app ls
```console
$ copilot app ls [flags]
```

## What does it do?

`copilot app ls` lists all the Copilot applications in your account.

## What are the flags?

```
-h, --help             help for ls
```

## Examples
List all the applications in your account and region.
```console
$ copilot app ls
```

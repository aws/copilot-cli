# app ls
```bash
$ copilot app ls [flags]
```

## コマンドの概要

`copilot app ls` は、AWS アカウントにある全ての Copilot Application を一覧表示します。

## フラグ

```bash
-h, --help             help for ls
```

## 実行例
AWS アカウント、リージョンにある全ての Application を一覧表示します。
```bash
$ copilot app ls
```
